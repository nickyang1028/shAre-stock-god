# Codex 项目指令

## 项目背景

这是一个股票 K 线信号自动识别项目。前端使用 TypeScript + React，后端使用 Node.js。核心能力是读取 K 线数据、计算技术指标、识别 K 线/指标信号，并在前端图表中展示。

## 开发原则

- 优先使用 TypeScript，前后端共享的数据结构要有明确类型。
- 信号识别逻辑保持纯函数，避免依赖 UI 状态或外部时间。
- 所有交易信号都作为分析提示，不写成投资建议。
- 新增规则时同步考虑测试样本、边界条件和前端展示字段。
- 不要把外部行情接口直接散落在业务代码里，统一放在 market-data 模块。

## 推荐模块边界

- `market-data`：行情接入、清洗、缓存、周期聚合。
- `indicators`：MA、EMA、MACD、KDJ、RSI、BOLL 等指标计算。
- `signals`：阳包阴、阴包阳、金叉、死叉等信号识别。
- `web`：K 线图、信号列表、筛选器、详情说明。
- `shared`：前后端共享类型、枚举和常量。

## 验证要求

- 修改信号识别规则时，补充或更新单元测试。
- 修改 API 契约时，同步更新共享类型和前端 service。
- 修改图表展示时，检查空状态、加载状态、错误状态和移动端布局。
