# Codex 模板说明

这个目录保存项目级 Codex 模板。

## skills

- `kline-signal-engineer`：用于设计和实现 K 线/指标信号识别规则。
- `stock-fullstack-builder`：用于开发 React + Node.js 的全栈功能。
- `market-data-integrator`：用于接入、清洗和归一化行情数据。

这些 skill 当前放在项目目录中，方便版本管理。若要让 Codex 自动发现，可以将需要启用的 skill 复制到 `~/.codex/skills/`。

## config.example.toml

这是 Codex 配置示例。按需复制或合并到 `~/.codex/config.toml`。

## AGENTS.md

仓库根目录的 `AGENTS.md` 用来记录本项目的长期协作约定。
